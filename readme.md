## datagen
Quick data generation tool - generates data as per format specification

### Dependencies
1. GNU C++ 11 compiler
2. GNU make
3. Boost 1.60

### Usage
```sh
./datagen <format-file>
```

to move to a file:
```sh
./datagen <format-file> > output.csv
```

> See format.json for example of a format file

### Make
- Modify Makefile for your boost directory and run `make`
